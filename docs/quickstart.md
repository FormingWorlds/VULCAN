# Quick start

!!! note "Usage within the PROTEUS framework"
    VULCAN can be used on its own, but is also integrated into the PROTEUS framework for the
    coupled evolution of rocky-planet atmospheres and interiors. Learn how to use PROTEUS
    [here](https://proteus-framework.org/PROTEUS).

## Fast track

Here is the quickest path to getting started:

1. **Install VULCAN**  
   Follow the installation steps and compile FastChem. <br>
   → [Installation guide](How-to/installation.md)
2. **Run your first model**  
   Run the default model, edit the configuration, and plot the results. <br>
   → [First-run tutorial](Tutorials/first_run.md)
