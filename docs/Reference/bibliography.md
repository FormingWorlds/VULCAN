# Publications

VULCAN has been described in the following two papers: 

- Tsai, S. M., Malik, M., Kitzmann, D., Lyons, J. R., Fateev, A., Lee, E., & Heng, K. (2021). A comparative study of atmospheric chemistry with VULCAN. _The Astrophysical Journal_, 923(2), 264. [https://doi.org/10.3847/1538-4357/ac29bc](https://doi.org/10.3847/1538-4357/ac29bc)

- Tsai, S. M., Lyons, J. R., Grosheintz, L., Rimmer, P. B., Kitzmann, D., & Heng, K. (2017). VULCAN: an open-source, validated chemical kinetics Python code for exoplanetary atmospheres. _The Astrophysical Journal Supplement Series_, 228(2), 20. [https://doi.org/10.3847/1538-4365/228/2/20](https://doi.org/10.3847/1538-4365/228/2/20)
